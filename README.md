Movie Recommendation System (C)

Overview

This project is a simple Movie Recommendation System developed using the C programming language.The program stores movie information and user ratings, and recommends movies to users based on their preferences.

Features

Stores movie details such as movie name and rating

Uses structures and arrays to manage data

Displays recommended movies for a specific user

Demonstrates basic recommendation system logic

Technologies Used

C Programming

Code::Blocks IDE

How the Program Works

Movie details and ratings are stored using structures.

User ratings for different movies are stored in arrays.

The program checks user ratings.

Movies with valid ratings are recommended and displayed in the console.

Example Output
